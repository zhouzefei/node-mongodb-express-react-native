# node-mongodb-express-react-native
node＋mongodb作为后端为react-native提供服务
#### node
##### 第一步：npm install 下载依赖资源
##### 第二步：启动mongodb服务
##### 第三步：node app.js

#### react-native
##### 第一步：npm install 下载依赖资源
##### 第二步：开启react-native(用xcode或者react-native run-ios启动)
